# Prokaiwa IG Character Bible

INTERNAL ONLY — these names are never written in any post, caption, or image.
(Note: this repo is the public website, so "internal" means "never published
in content," not "secret." The names are deliberately low-stakes.)

## P-kun — `p(´∇｀)q`
Solo mascot for **Word of the Day** (Mon) and **Phrase of the Day** (Wed) cards.
His hands are literally a lowercase **p** and **q** — P for Prokaiwa. Always
cheering the learner on. Energetic, warm, never sarcastic.

| Mood | Pose |
|---|---|
| cheering (signature) | `p(´∇｀)q` |
| normal | `p(・ω・)q` |
| determined | `p(｀・ω・´)q` |
| surprised | `p(ﾟoﾟ)q` |
| sleepy | `p(-ω-)q` |
| oops / sad | `(p_q)` |

## Mimi — `૮₍˶ •. • ⑅₎ა`
The **boke** (silly one) of the Friday dialogue carousel. Has little ears and a
bow-tie cheek `⑅`. Sweet, earnest, slightly absurd logic. ALWAYS delivers the
punchline on the final slide with her most expressive face. Loves food,
naps, and over-literal interpretations of English.

| Mood | Pose |
|---|---|
| neutral | `૮₍˶ •. • ⑅₎ა` |
| happy | `૮₍˶ ᵔ ᵕ ᵔ ⑅₎ა` |
| shocked | `૮₍˶ ⊙ o ⊙ ⑅₎ა` |
| smug | `૮₍˶ ˘ ³˘ ⑅₎ა` |
| dizzy / defeated | `૮₍˶ ×. × ⑅₎ა` |

## Ten — `(„• ֊ •„)`
The **tsukkomi** (straight man). Little arms `„ „`. Polite, patient, asks the
normal questions that set Mimi up. Reacts to the punchline with maximum
shock or deadpan resignation on the final slide.

| Mood | Pose |
|---|---|
| neutral | `(„• ֊ •„)` |
| happy | `(„ᵔ ֊ ᵔ„)` |
| shocked | `(„⊙ ֊ ⊙„)!!` |
| deadpan | `(„￣ ֊ ￣„)` |
| defeated | `(„x ֊ x„)` |

## Format rules
- Carousel = 5 slides: title card → 3 exchange slides (Ten usually sets up,
  Mimi answers) → punchline slide (Mimi's absurd line + Ten's big reaction),
  ending with the key phrase recap + P-kun cheer is allowed on a recap line.
- Comedy register: gentle, wholesome silliness. Food jokes, nap jokes,
  taking idioms literally. Never mean, never edgy, no romance.
- Every dialogue teaches ONE real, useful English phrase (the title phrase).
- Card design: cream `#FFF8F0` background, teal `#008080` frame + title,
  dark `#333` text, Prokaiwa P logo chip (circle) bottom-right. All bubble /
  box lines stay INSIDE the teal frame.
- Caption: phrase EN/JP + one-line usage note + CTA + hashtags. No character
  names anywhere.
